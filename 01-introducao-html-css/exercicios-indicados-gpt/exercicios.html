Exercícios de HTML

1. Exercício com a tag <img>

Objetivo: Melhorar a acessibilidade e organização das imagens.

Tarefa:

  • Adicione legendas descritivas às imagens utilizando a tag <figcaption>.
  • Melhore a acessibilidade garantindo que o atributo alt seja descritivo e adequado.
  • Experimente utilizar a tag <picture> para adaptar imagens a diferentes tamanhos de tela.

2. Exercício com a tag <a>

Objetivo: Aprimorar a navegação e interatividade.

Tarefa:

  • Adicione um botão estilizado para o link do WhatsApp utilizando <button>.
  • Crie um menu de navegação fixo no topo da página utilizando <nav> e links internos.
  • Adicione ícones aos links usando a tag <img> ou bibliotecas como FontAwesome.

3. Exercício com a tag <div> e estruturação de conteúdo

Objetivo: Melhorar a organização visual da página.

Tarefa:

  • Utilize <section> para dividir as diferentes partes do site de forma mais semântica.
  • Aplique classes e IDs às <div> para facilitar a estilização com CSS.
  • Experimente agrupar elementos relacionados em um <article> ou <aside>, se fizer sentido.
